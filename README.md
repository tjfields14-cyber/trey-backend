# Pearl.Ai — Clean Deployment Bundle
Version: Tauras 1

This deployment bundle reconstructs the current Pearl.Ai runtime structure
for continued widget + daemon integration work.

## Included Systems
- Express backend runtime
- WebSocket synchronization layer
- Runtime heartbeat
- Narrative pulse emitter
- Reflection loop
- Executive HUD scaffold
- Frontend StateManager
- React/Vite frontend shell

## Runtime Ports
Backend: http://localhost:4000
Frontend: http://localhost:5173

## Startup

Backend:
npm install
npm start

Frontend:
cd frontend
npm install
npm run dev
