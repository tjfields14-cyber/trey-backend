- Returns all chapters sorted by chapter_number, id.

Chapter summaries
-----------------
POST /chapters/:id/summary
- Loads chapter by id.
- Builds a summarization prompt (dark fantasy / horror tone, 1–2 paragraphs, styleHint optional).
- Calls OpenAI and stores result in chapters.summary.
- Returns updated chapter and summary.

GET /chapters/:id/summary
- Returns { id, chapter_number, title, summary } only.

Chapter AI edits
----------------
POST /chapters/:id/edit
- Loads chapter.
- Builds line-edit prompt:
  - Keep voice, POV, tense
  - Keep all events and worldbuilding
  - Focus on clarity and rhythm
  - Always return a fully written chapter, not notes.
- Calls OpenAI.
- Inserts into chapter_edits (instructions, edited_text).
- Returns the new edit row.

GET /chapters/:id/edits
- Lists all edits for a chapter (id, chapter_id, instructions, created_at).

GET /edits/:editId
- Returns a specific edit (full row, including edited_text).

Merge edit into chapter (with snapshot)
---------------------------------------
POST /chapters/:id/merge-edit
Body: { editId, label }

Logic:
1) Connect a Postgres client and BEGIN a transaction.
2) Load chapter (by chapterId) and edit (by editId).
3) Verify edit.chapter_id matches chapterId.
4) Insert a snapshot into chapter_versions including previous text, summary, word_count, and label ("pre-merge from edit X" or your custom label).
5) Calculate new word_count from edit.edited_text.
6) UPDATE chapters SET
   - text = editedText
   - word_count = new count
   - merged_edit_id = editId
   - updated_at = NOW()
7) COMMIT and return updated chapter.

--------
GET /admin
- Renders an HTML page that lists all chapters in a simple table:
  - # (chapter_number)
  - Title
  - Words
  - Summary? (✔️ or —)

6. POWERSHELL: ASK-TREY HELPER
==================================================

On your Windows machine, you defined a PowerShell function Ask-Trey, along these lines:

function Ask-Trey {{
  param(
    [Parameter(Mandatory = $true)]
    [string]$Question
  )

  $body = @{
    source   = "trey-widget"
    question = $Question
  } | ConvertTo-Json

  $response = Invoke-RestMethod -Uri "http://192.168.137.2:3000/kb/ask" `
    -Method POST `
    -ContentType "application/json" `
    -Body $body

  return $response
}}
